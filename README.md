# Modern-Personal-Portfolio
A modern, responsive personal portfolio with a CSS-based dark mode and clean UI design.
## Features

* **Full Responsive Design:** Works perfectly on desktops, tablets, and mobile devices.
* **Dark Mode Toggle:** Integrated dark and light mode switching using pure CSS logic.
* **Modern UI:** Built with Google Fonts (Poppins) and interactive icons from Boxicons and FontAwesome.
* **Smooth Navigation:** Includes internal links with smooth scrolling behavior.

## Technologies Used

* **HTML5:** For the structural layout.
* **CSS3:** Custom properties (Variables), Flexbox, and CSS Grid.
* **Icon Libraries:** FontAwesome and Boxicons for visual representation.

## File Structure

* `index.html`: The main file containing both the structure and the styling.
* `photo.png`: The personal profile image displayed in the home section.
## Layout Details

The project follows a card-based UI where each section is encapsulated in a "custom-card" class, providing a clean separation of concerns and a consistent look across the portfolio.
